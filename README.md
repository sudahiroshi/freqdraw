# freqdraw

## Introduction
マイクから入力した音声波形とそのスペクトルを表示します．

## Usage

画面下部にある"FFT"ボタンをクリックすると，音声入力を開始します．

"Line"ボタンをクリックすると，マウス操作により，スペクトルのグラフ上に基音や倍音を表示します．クリック後にスペクトルのグラフ上でマウスを動かしてください．

"Pen"ボタンをクリックすると，マウスドラッグにより，スペクトルのグラフ上に描画できます．
